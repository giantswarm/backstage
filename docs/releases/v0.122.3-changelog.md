# Release v0.122.3

## @giantswarm/backstage-plugin-ai-chat@0.11.6

### Patch Changes

- f800ea6: Reasoning blocks in the AI chat now stream live and look like other modern
  chat UIs (ChatGPT, Claude, Cursor).

  Previously the `ReasoningGroup` slot defaulted to **closed**, so even though
  the SDK was streaming `reasoning-delta` chunks every ~50-200ms (verified
  end-to-end against vLLM/Nemotron), the user only saw collapsed "Reasoning"
  triggers pop into the chat -- creating the perception that everything
  happened "at once" at the end.

  Changes:
  - The reasoning block now **auto-opens while the model is reasoning** and
    **auto-collapses** ~800ms after the model moves on to a tool call or the
    final answer. The user can override either way with a click.
  - The trigger label is now a live progress indicator: "Thinking 5s…" while
    streaming, "Thought for 12s" once finished.
  - Reasoning text reveals character-by-character (same animator used by
    `StreamingMarkdownText`) so the user perceives smooth typing even when SSE
    chunks are batched into a single React render.
  - The reasoning container auto-scrolls to the latest tokens while streaming.
  - Visual redesign: borderless block with a subtle 2px left accent line that
    highlights in the primary color while streaming, italic muted text, and a
    small inline trigger -- matching the "thinking" presentation in modern
    chat UIs instead of the previous heavy boxed look.
  - The global "Thinking…" spinner below the message is now hidden once any
    assistant content (reasoning or text) starts streaming, so the in-message
    Reasoning trigger and the streaming text itself act as the single live
    progress indicator.
