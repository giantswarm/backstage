## General

- The offical Giant Swarm documentation website is https://docs.giantswarm.io.
- The documentation source is available on GitHub at https://github.com/giantswarm/docs/tree/main/src/content in Markdown format.
- For example, the page https://github.com/giantswarm/docs/blob/main/src/content/overview/security/domain-allowlist/index.md is pubslished as https://docs.giantswarm.io/overview/security/domain-allowlist/
- An index of most documentation pages with title, description, and URL is available via https://docs.giantswarm.io/llms.txt in Markdown format. The Changes & Releases part of docs is not included in this index.
- The URL structure is strictly hierarchical, with path segments indicating main sections, sub-sections and individual pages.

## Main sections

The documentation website is structured into the following main sections:

- [Overview](https://docs.giantswarm.io/overview/): high-level information

- [Getting started](https://docs.giantswarm.io/getting-started/): introductory information for new users

- [Tutorials](https://docs.giantswarm.io/tutorials/): step-by-step guides for common tasks

- [Reference](https://docs.giantswarm.io/reference/): in-depth technical reference material

- [Support and training](https://docs.giantswarm.io/support/): how Giant Swarm provides support and training

- The [Changes & Releases](https://docs.giantswarm.io/changes/) section is an aggregation of important changelogs and release notes. There is an [RSS feed](https://docs.giantswarm.io/changes/index.xml) available, to subscribe to releases. Sub-categories have their own feeds.
